# SillyTavern 导入角色和世界书步骤

## 📥 导入角色卡

### 方法一：通过角色管理界面导入

1. **打开角色管理**
   - 点击左侧的 **"角色"** 按钮（人物图标）
   - 或者点击左下角的 **"Characters"**

2. **导入角色**
   - 点击右上角的 **"导入"** 按钮（向下箭头图标）
   - 选择 **"从文件导入"**
   - 导航到：`SillyTavern/data/default-user/characters/`
   - 选择你想导入的角色文件夹（如：彩叶、八千代、辉夜等）
   - 选择文件夹内的 `.json` 文件

3. **批量导入**
   - 可以一次选择多个角色的 JSON 文件
   - 系统会自动导入所有选中的角色

### 方法二：直接复制（已自动完成）

✅ **好消息**：角色卡已经自动放在正确的位置了！
- 位置：`SillyTavern/data/default-user/characters/`
- 包含31个角色，每个角色都有独立的文件夹

你只需要刷新角色列表就能看到它们。

## 📚 导入世界书

### 步骤：

1. **打开世界书管理**
   - 点击顶部的 **"世界信息"** 按钮（地球图标 🌍）
   - 或者点击 **"World Info"**

2. **导入世界书**
   - 点击 **"导入世界书"** 按钮
   - 导航到：`SillyTavern/data/default-user/worlds/`
   - 选择 `超时空辉夜姬.json`
   - 点击确认

3. **启用世界书**
   - 导入后，在世界书列表中找到 "超时空辉夜姬"
   - 点击 **"启用"** 或 **"激活"**
   - 确保状态显示为 **"已启用"**

---

## 🎭 创建群组聊天（关键步骤）

导入角色和世界书后，你需要创建群组来让角色互相对话：

### 步骤：

1. **进入群组管理**
   - 点击左侧的 **"Group Chats"** 按钮（多人图标）
   - 或者点击 **"群组聊天"**

2. **创建新群组**
   - 点击 **"Create Group"**（创建群组）
   - 填写群组名称：例如 "超时空辉夜姬 - 日常篇"

3. **选择成员**
   - 在成员列表中勾选你想要的角色：
     - ✅ 彩叶
     - ✅ 八千代
     - ✅ 辉夜
     - （其他你想加入的角色）

4. **配置生成模式**
   - **Generation Mode**: 选择 **"SWAP"**（自动切换角色）
   - **Activation Strategy**: 选择 **"NATURAL"**（自然选择）
   - **Auto Mode Delay**: 设置为 **5** 秒

5. **保存群组**
   - 点击 **"Save"** 或 **"创建"**

---

## 🚀 开始生成小说

### 步骤：

1. **进入群组聊天**
   - 点击你创建的群组卡片
   - 进入聊天界面

2. **加载世界书**
   - 确保世界书 "超时空辉夜姬" 已启用
   - 点击世界书图标检查

3. **开始对话**
   - 在输入框输入开场白（可选）
     - 例如："今天天气真好，我们去海边吧！"
   - 或者留空，让角色自己开始

4. **生成内容**
   - 点击 **"Generate"** 按钮
   - 角色会自动开始对话

5. **自动模式（推荐）**
   - 点击 **"Auto Mode"** 按钮
   - 设置延迟时间（5-10秒）
   - 角色会自动持续对话，生成小说内容！

## ❓ 常见问题

**Q: 角色没有显示在列表中？**
A: 点击刷新按钮，或重启 SillyTavern。

**Q: 世界书没有生效？**
A: 确保在群组聊天界面中启用了世界书。

**Q: 角色不说话？**
A: 检查 API 是否连接成功，以及是否有足够的 API 额度。

**Q: 如何导出生成的小说？**
A: 点击聊天界面的导出按钮，可以导出为文本文件。
