# 超时空辉夜姬 - SillyTavern 使用指南

## ✅ 已完成的工作

你的知识库已经成功转换为SillyTavern格式：

- **31个角色卡**：每个角色都有完整的设定（性格、说话风格、人际关系）
- **1个世界书**：包含世界观、核心概念、重要地点等设定

## 🚀 如何开始使用

### 第一步：启动 SillyTavern

```bash
cd SillyTavern
node server.js
```

然后在浏览器打开：`http://localhost:8000`

### 第二步：导入角色

1. 点击左侧的 **"Character Management"**（角色管理）
2. 点击 **"Import"**（导入）
3. 选择角色文件夹（例如：`彩叶`）
4. 点击导入

### 第三步：配置API

**推荐使用 DeepSeek API**（便宜且质量好）：

1. 点击右上角的 **"API"** 按钮
2. 选择 **"OpenAI"** 或 **"OpenRouter"**
3. 填入API密钥：
   - DeepSeek API Key: `sk-xxxxxxxx`
   - Base URL: `https://api.deepseek.com/v1`
   - Model: `deepseek-chat`

**或者使用其他API**：
- OpenAI: `https://api.openai.com/v1` + `gpt-4o-mini`
- Moonshot: `https://api.moonshot.cn/v1` + `moonshot-v1-8k`

### 第四步：加载世界书

1. 点击 **"World Info"**（世界信息）
2. 点击 **"Import"**（导入）
3. 选择 `超时空辉夜姬.json`
4. 确保世界书已启用

### 第五步：开始对话！

选择一个角色，开始聊天吧！
