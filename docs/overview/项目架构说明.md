# NovelForge 项目架构说明

> 最后更新：2026-03-20 | 版本：v1.2

---

## 项目概述

NovelForge 是一个基于 Web 的小说创作辅助工具，采用前后端分离架构：
- **前端**：Next.js 15 (App Router) + TypeScript
- **后端**：Python FastAPI
- **外部集成**：SillyTavern API（AI 对话生成）

核心流程：**导入小说/文本 → 提取核心要素 → AI 对话创作 → 生成新剧情**

```

│   ├── frontend/               # Next.js 前端

│   ├── novelforge/             # Python 后端包
│   │   ├── api/                # FastAPI 端点（main.py, ai_planning_service.py, text_processing.py）
│   │   ├── extractors/         # 文本提取器（角色、关系、时间线、世界设定）
│   │   ├── content/            # 内容管理模块
│   │   ├── quality/            # 质量控制模块
│   │   ├── services/           # 业务服务层
│   │   ├── storage/            # 存储层
│   │   ├── types/              # 类型定义
│   │   └── world_tree/         # 世界树结构
│   ├── scripts/                # 工具脚本
│   └── docs/                   # 前端组件文档（LAYOUT_SYSTEM, ERROR_HANDLING, FORM_ENHANCEMENT）
├── SillyTavern/                # SillyTavern 集成
```

## 快速启动

```bash
# 后端
cd novelforge-core
python -m venv .venv && .venv\Scripts\activate
pip install -e .
uvicorn novelforge.api.main:app --reload --host 0.0.0.0 --port 8000

# 前端
cd novelforge-core/frontend
npm install
npm run dev
```

访问：
- 前端：http://localhost:3000
- 后端 API 文档：http://localhost:8000/docs
