# NovelForge 生成质量评估 (Quality) 模块逻辑架构文档

因为 AI 生成和提取的文本可能是不稳定的（甚至可能存在大模型幻觉），所以本目录承担了“监督系统”的职责。负责给产出的内容结果打分定级。

## 1. 内部代码内联逻辑图 (Internal Logic Flow)

```text
[ 获取大模型产出的最终数据 ]
          |
  [ evaluator.py 评估系统 ]  <---> [ scorer.py 评分函数库 ]
          |
    (计算信息重叠度、格式合规性、是否漏提、是否出现幻觉)
          |
  [ 评估分数和指标反馈给上层进行决断 (重新提取或通过) ]
```

## 2. 核心文件职责说明

| 文件名 | 职责描述 |
| :--- | :--- |
| **`evaluator.py`** | 作为上层接口，调用底层的计分系统进行全项指标评定，并给出最终建议。 |
| **`scorer.py`** | 定义了具体如何计算“分数”的规则。比如对比提取的角色名与原文出现的比率，对比上下文关联度。 |
