# NovelForge 后端指令行 (CLI) 模块逻辑架构文档

本目录作为 `novelforge-core` 纯脱离 Web 环境时的命令行工具入口。通过它可以直接在终端处理大量批量任务。

## 1. 内部代码内联逻辑图 (Internal Logic Flow)

```text
[ 开发者 / 自动化终端命令 ] --(传递参数)--> [ main.py (CLI 接驳口) ]
                                             |
                                             v
                             +-----------------------------------+
                             |       Services 侧业务实现代码       |
                             +-----------------------------------+
```

## 2. 核心文件职责说明

| 文件名 | 职责描述 |
| :--- | :--- |
| **`main.py`** | 基于现代 CLI 库 (如 Click/Typer) 构建，对外提供 `novelforge extract ...` 之类的直接调用入口，方便离线数据处理与调试而不必启动前端。 |
